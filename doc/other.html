Ostatní funkce

JAX.makeClass( NAME [, EXTEND ] )

Obalení JAK.ClassMaker.makeClass. NAME je string s názvem třídy a EXTEND je případná třída z níž se bude dědit.

Příklad

Trida = JAX.makeClass( "Trida" )
Trida = JAX.makeClass( "Trida", TridaZNizDedim )

JAX.make( def [, attrs, doc, nodecor ] )

Funkce vytvoří element dle zadané "CSS query" def. Elementu je možno nastavit další vlastnosti attrs (jde od asoc. pole). Dále je možné určit dokument (doc) v jehož kontextu se element vytvoří.

Příklad

JAX.make( "div.trida1.trida2" )// vystupem je <div class="trida1 trida2"/>
JAX.make( "div#idecko.trida1" )// vystupem je <div id="idecko" class="trida1"/>
JAX.make( "div", { "title": "Popis" } )// vystupem je <div title="Popis" alt="Popis"/>

Seznam možných vlastností (attrs)

Název Hodnota Co dělá
htmlHTMLVloží do elementu html (zkratka pro innerHTML)
addhtmlHTMLPřidá do elementu html (zkratka pro innerHTML+=...)
positionabs/rel/relative/absolutenastavuje style.position
widthčíslo (1) nebo string ("1px")nastavuje style.width (pokud je zadáno číslo použijí se PX)
heightčíslo (1) nebo string ("1px")nastavuje style.height (pokud je zadáno číslo použijí se PX)
leftčíslo (1) nebo string ("1px")nastavuje style.left (pokud je zadáno číslo použijí se PX)
topčíslo (1) nebo string ("1px")nastavuje style.top (pokud je zadáno číslo použijí se PX)

Dále je mozno zadat různé události (vypsáno z JAX._elmEvents):

A teoreticky jakoukoli jinou vlastnost elementu. Např.: target, href, name, src ...

Příklad

JAX.make( "div", { "click": function(e, elm){ ... } } );
JAX.make( "div", { "width": "160px" } );
JAX.make( "a", { "href": "/path/to" } );
JAX.make( "img", { "src": "naha_baba.png" } );

JAX.OffScreenMeter( elm, [ meter_in_dom ] )

Změří rozměry elementu bez nutnosti jeho appendování do dokumentu. Fce funguje tak, že vytvoří kopii elementu a vloží ji jako první element v body, poté se "přečtou" hodnoty offsetWidth a offsetHeight.
Vrací: {width:#,height:#}

Verze 1.15 Pokud je zadán volitelný argument meter_in_dom pak se vytvořená kopie elementu nevkládá jako první element v body, ale před stávajicí element - tudíže lze takto měřit jen elementy již vložené do dokumentu (neplatí tedy první věta dokumentace této fce). Výhodou je, že na element jsou aplikována CSS pravidla odpovídajicí jeho umístění v dokumentu.

třída JAX.Delay( func, time [, ... ] )

Funguje jako klasický setTimeout jen jde o třídu a volané funkci lze předat libovolné argumenty.
Má metodu stop() jež naprosto nečekaně zruší probíhající časovač.
Více v příkladu.

třída JAX.Interval( func, time [, ... ] )

Funguje jako klasický setInterval jen jde o třídu a volané funkci lze předat libovolné argumenty.
Má metodu stop() jež naprosto nečekaně zruší probíhající časovač.
Více v příkladu.

Příklad: interval & delay

Příklad je uveden jen pro JAX.Delay, ale platí i pro JAX.Interval.

function funkce1( a1, a2 ){ ... }
// po uplynutí 200ms zavolá fci funkce1 s argumenty "x", "y"
var timer = new JAX.Delay( funkce1, 200, "x", "y" );
timer.stop() // ... nezavolá jelikož jsme časovač zrušili
