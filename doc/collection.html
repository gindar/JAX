Kolekce elementů

JAX.$( query, [ context, single ] )

Tato funkce vyhledá v aktuálním dokumentu ( nebo v rámci nějakého elementu context ) elementy vyhovující zadané CSS1 query.

Pokud je zadán argument single == true pak funkce vrátí jeden (první) element jež je odekorován. V opačném případě je vrácena instance JAX.HTMLCollection.

Příklad

JAX.$( "div#box div a.link" )
JAX.$( "div a, span a, ul li a" )

JAX.HTMLCollection

Toto je třída jež uchovává seznam elementů a stará se o hromadné volání metod elementu. Při vytvoření kolekce se všechny její prvky odekorují.
Instance HTMLCollection má několik užitečných vlastností:
    <instance>.elements - pole elementů jež kolekce obsahuje
    <instance>.length - počet elementů jež kolekce obsahuje
    <instance>.debug - obsahuje textovou reprezentaci obsahu kolekce; generuje se jen pokud je JAX.DEBUG == true

JAX.HTMLCollection.prototype.push( ... )

Vloží do kolekce elementy jež jsou zadány jako jednotlivé argumenty.

Příklad

collection.push( element, element2 )

JAX.HTMLCollection.prototype.pop( )

Vyjme z kolekce poslední element a vrátí jej.

JAX.HTMLCollection.prototype.shift( )

Vyjme z kolekce první element a vrátí jej.

JAX.HTMLCollection.prototype.slice( startIndex [, end ] )

Vrací "výřez" z kolekce. Výřez je nová JAX.HTMLCollection. Původní kolekce zůstává nezměněna.

JAX.HTMLCollection.prototype.splice( startIndex [, count] )

Vyjme "výřez" (odkud - startIndex a kolik - count) z kolekce a vrátí jej. Výřez je nová JAX.HTMLCollection. Původní kolekce se změní.
